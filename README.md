How to Run the Project Locally
Requirements
  Windows OS

  Visual Studio 2022 (or newer)

  .NET Desktop Development workload installed

  SQL Server or SQL Server Express

  SQL Server Management Studio (SSMS)

Steps

  Open the project by selecting the .sln file in Visual Studio.

  Create a local database in SQL Server (e.g., LogisticsSystemDB).

  Run the provided SQL scripts to create the required tables.

  Update the connection string in the project to match your local SQL Server setup.

  Run the project by pressing F5 in Visual Studio.
